V1
The model uses a standard **encoder-decoder Transformer** (Vaswani et al., 2017 — "Attention Is All You Need"), with a few adaptations for sign language:

**Standard components kept as-is:**
- Multi-head self-attention + cross-attention
- Positional encoding (sinusoidal)
- Feed-forward sublayers
- Autoregressive decoding with causal mask

**Adaptations made for this task:**

| Design choice | What was done | Why |
|---|---|---|
| Input | Linear projection + LayerNorm + ReLU instead of token embedding | Continuous keypoint vectors, not discrete tokens |
| Normalisation | Pre-LN (norm before attention) | More stable training than original post-LN |
| Output | Standard token softmax over English vocabulary | Same as NMT |
| Positional encoding | Sinusoidal on both encoder and decoder | Standard |
| Decoding | Greedy + beam search with length penalty | Standard NMT practice |

**What it is NOT:**
- Not a vision transformer (ViT) — no raw video/pixels
- Not a graph neural network over the keypoint skeleton
- Not a CTC-based model (common in some SLR work)
- Not pretrained (no transfer from mBERT, mBART etc.)

**Closest published work it resembles:**
- Sign Language Transformers (Camgoz et al., 2020) which introduced the encoder-decoder transformer for sign language translation.
